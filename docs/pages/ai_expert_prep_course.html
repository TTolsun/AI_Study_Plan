AI EXPERT PREP COURSE

AI Expert 시험 대비반

AI Expert 시험 합격을 목표로 Python 기초, 딥러닝·컴퓨터비전 기초, Transformer/LLM 입문 내용을 압축 학습합니다. 전체 일정은 강의 8회와 최종 모의고사 1회로 구성하고, Python 범위 점검과 딥러닝·CV·LLM 범위 점검은 별도 회차가 아니라 해당 강의 마지막 20분을 활용해 진행합니다.

운영 원칙

원칙 1

사전 학습 후 세션에서 압축 정리

링크 자료와 영상은 세션 전에 보고 들어오는 것을 기본으로 합니다. 세션에서는 시험에 나올 만한 개념, 헷갈리는 비교 포인트, 계산 문제를 중심으로 정리합니다.

원칙 2

강의 중 예상문제 풀이 포함

각 강의 회차마다 예상문제 1~3개를 함께 풉니다. 단순히 내용을 들었다는 상태에서 끝나지 않도록, 이해가 부족한 지점을 빠르게 확인합니다.

원칙 3

범위 점검은 강의 마지막 20분 활용

Python 범위 점검과 딥러닝·CV·LLM 범위 점검은 별도 회차로 잡지 않습니다. 각각 Python 마무리 회차와 전체 이론 마무리 회차의 마지막 20분을 사용합니다.

기본 운영안은 총 9회입니다. 강의 8회와 최종 모의고사 1회로 구성하며, 1차 Python 범위 점검과 2차 딥러닝·CV·LLM 범위 점검은 강의 시간 안에서 20분씩 진행합니다.

90분 회차 운영 방식

모든 강의 회차를 같은 방식으로 운영하지 않고, 일반 강의 회차와 범위 점검 포함 회차를 구분합니다. 시험 대비반의 목적은 단순한 진도 소화가 아니라, 실제 시험에서 취약해질 수 있는 지점을 미리 확인하고 보완하는 것입니다.

일반 강의 회차

개념 정리 + 예상문제

핵심 개념 정리 55분, 예상문제 풀이 25분, 오답·다음 과제 정리 10분으로 운영합니다.

범위 점검 포함 회차

강의 70분 + 점검 20분

Session 02와 Session 08은 마지막 20분을 범위 점검으로 사용합니다. 문제 수는 3~5문항 수준으로 제한하고, 오답은 다음 회차 시작 시 짧게 복습합니다.

최종 모의고사 회차

전체 범위 실전 점검

마지막 회차는 별도 모의고사로 운영합니다. 전체 범위를 섞어 풀고, 개인별 오답노트와 최종 암기 카드를 정리합니다.

AI Expert 대비 커리큘럼

Python을 먼저 정리한 뒤 Python 범위 점검을 강의 마지막 20분에 진행합니다. 이후 CS231n과 CME295를 학습하고, 전체 이론 마무리 회차에서 딥러닝·CV·LLM 범위 점검을 20분 진행합니다. 마지막에는 별도 최종 모의고사 1회로 전체 범위를 점검합니다.

Session 01 · Lecture
Python 핵심 문법과 코드 읽기
Python Syntax Code Reading
학습 내용

Python 기본 실행 흐름, 숫자/문자열, list·tuple·dict·set, 조건문, 반복문, slicing, comprehension을 시험용 코드 해석 관점으로 정리합니다.

예상문제 방향

list slicing 결과, dict/set 특징, 반복문 실행 결과, mutable/immutable 구분, comprehension 결과 예측

Session 02 · Lecture + Check
Python 함수·모듈·예외·파일 처리 + 1차 범위 점검
Function Module 20min Check
학습 내용

함수 인자, scope, lambda, 모듈 import, package 개념, 예외 처리, 파일 I/O, class 기본을 정리합니다. 마지막 20분은 Python 범위 점검에 사용합니다.

점검 방향

default argument, import 방식, try/except/finally 실행 순서, 파일 읽기 흐름, class attribute와 instance attribute 차이

Session 03 · Lecture
CS231n Module 1: Classification과 학습 기초
kNN Linear Classifier Loss
학습 내용

Image classification, kNN, train/validation/test split, linear classifier, SVM hinge loss, softmax cross-entropy, regularization을 하나의 흐름으로 정리합니다.

예상문제 방향

k값 변화 영향, validation set 목적, loss function 구분, regularization 효과, overfitting/underfitting 판단

Session 04 · Lecture
CS231n Module 1~2: Optimization, NN, CNN
SGD Backprop CNN
학습 내용

Gradient descent, learning rate, backpropagation, activation function, dropout, batch normalization, convolution layer, pooling, stride, padding, receptive field를 정리합니다.

예상문제 방향

gradient check 목적, learning rate 문제, dropout/BatchNorm 효과, CNN 출력 feature map 크기 계산, parameter 수 계산

Session 05 · Lecture
CS231n Module 2: CNN 해석과 Transfer Learning
Visualization Transfer Fine-tuning
학습 내용

CNN visualization, feature embedding, architecture별 특징, transfer learning, freezing/fine-tuning 전략을 정리합니다. Camera 업무 관점에서는 모델이 이미지를 어떤 feature로 보는지 감을 잡는 구간입니다.

예상문제 방향

feature visualization 목적, 데이터가 적을 때의 학습 전략, fine-tuning/freezing 차이, CNN 구조 요소별 역할

Session 06 · Lecture
CME295 Lecture 1: Transformer 기본 구조
Tokenization Attention Transformer
학습 내용

Tokenization, embedding, word2vec, RNN/LSTM 한계, attention mechanism, Transformer architecture를 LLM 입문 관점으로 정리합니다.

예상문제 방향

RNN/LSTM 한계, attention이 필요한 이유, embedding 의미, Transformer block 구성, self-attention의 입력·출력 관계

Session 07 · Lecture
CME295 Lecture 2: Attention 심화와 BERT 계열
MHA/MQA/GQA RoPE BERT
학습 내용

Attention approximation, MHA/MQA/GQA, positional embedding, RoPE, Transformer-based architecture, BERT 계열 개념을 비교 중심으로 정리합니다.

예상문제 방향

Position embedding 역할, MHA/MQA/GQA 차이, RoPE 목적, BERT 계열 특징, Encoder/Decoder 구조 비교

Session 08 · Lecture + Check
딥러닝·CV·LLM 종합 정리 + 2차 범위 점검
Review Non-Python 20min Check
학습 내용

CS231n과 CME295에서 헷갈리기 쉬운 loss, optimization, CNN 계산, attention 구조, Transformer 계열 비교를 다시 묶어 정리합니다. 마지막 20분은 딥러닝·CV·LLM 범위 점검에 사용합니다.

점검 방향

CNN 출력 크기와 parameter 수 계산, loss·regularization 비교, MHA/MQA/GQA 차이, RoPE 목적, Encoder/Decoder 구조 비교

Session 09 · Final Mock
최종 모의고사: 전체 범위 실전 점검
Final Mock Full Range Final
진행 방식

Python, CS231n, CME295 범위를 모두 섞어 실제 시험처럼 시간 제한을 두고 풉니다. 문제는 개념형, 계산형, 코드해석형을 혼합합니다.

마무리

개인별 오답노트를 정리하고, 최종 암기 카드와 자주 틀린 문제 목록을 남깁니다.
